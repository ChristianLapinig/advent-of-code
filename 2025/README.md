# Advent of Code Solutions

## Overview
This is a compilation of solutions for Advent of Code.

## Contributing
Please note, even though you are free to create a pull request, 
I will not be actively accepting them as this repository is meant to help
others and myself. If you have suggestions or optimizations, 
please add them to the "Issues" tab. If I accept your suggestion, I will
do my best to give you credit. Likely in the form of a comment in my code.

### Cloning and Running the Code
```bash
# Clone the repository
$ git clone https://github.com/ChristianLapinig/advent-of-code.git
$ cd advent-of-code

# Change directory into the year of the solution (2025 in this example)
$ cd 2025

# Change into the day
$ cd day1

# Run main.go
$ go run main.go input.txt # Just an example. I might have files named differently
```
